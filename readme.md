This project is a simple customer records management system built using JavaScript. It stores and manages customer information such as names, emails, and purchase history using arrays and objects.
Technologies Used:
JavaScript
VS Code
What I Learned
From this project, I learned how to work with arrays and objects together to store structured data. I also practiced adding and removing data using methods like .push() and .shift(). In addition, I learned how to update object properties and loop through data using .forEach().
